## Profiles

- `exhausted_patient.csv`: ผู้ป่วยหมดแรง นอนเป็นหลักแทบทั้งวัน มีลุกนั่งหรือเดินสั้นมากเฉพาะจำเป็น
- `dementia_night_wandering.csv`: ผู้ป่วยมีภาวะสมองเสื่อม เดินมากผิดปกติช่วงกลางคืน กลางวันมีพักนอนและนั่งบ้าง
- `depressed_sedentary.csv`: ผู้ป่วยมีภาวะเศร้า นั่งเป็นหลักเกือบทั้งวัน เคลื่อนไหวน้อยมาก และเข้านอนค่อนข้างตรงเวลา
- `insomnia_restless_night.csv`: ผู้ป่วยนอนไม่หลับ ลุกขึ้นลงสลับกันหลายครั้งตอนกลางคืน กลางวันยังมีการนั่งและเดินได้บ้าง
- `normal_sleep_cycle.csv`: ผู้ป่วยนอนหลับดี กลางคืนนอนต่อเนื่อง กลางวันมีนั่ง เดิน และออกจากพื้นที่บางช่วง

## Action Labels

- `no_person`: ไม่มีคนอยู่ในพื้นที่ที่กล้องหรือเซนเซอร์มอนิเตอร์
- `standing`: ยืนอยู่กับที่
- `walking`: กำลังเดินหรือเคลื่อนที่
- `sitting`: กำลังนั่ง
- `lying`: กำลังนอนหรือนอนราบ
- `get_up`: ช่วงเปลี่ยนจากนั่ง/นอนเป็นลุกขึ้น
- `get_down`: ช่วงเปลี่ยนจากยืนเป็นนั่งหรือนอน
